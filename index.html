KronoStory

A platform for passion projects.

Live GitHub
Overview

KronoStory is a community platform for finding and showcasing creative work and personal endeavors, featuring timelines that make it easy to see the whole process. Users can sign up and log in to share their projects. They can add text and images, update their projects, and delete their content. Visitors to the site can filter content by categories and view user profiles.

Technical Details

Utilized JavaScript with React.js to build the appication's front end UI. Also learned React Router and Axios to ensure proper page routing and to send/retrieve data from endpoints. Back-end built with PHP to make queries that save project content in a MySQL database. Project features custom CSS3 and Bootstrap 4 for a responsive layout. All code, data, and user-generated content are hosted on an AWS EC2 server.

Dozen's Bakery

An online storefront for a bakery (demo only).

Live GitHub
Overview

An online storefront for a bakery that sells treats by the dozen. Visitors can browse the site, add and remove treats from their cart, place an order, and view their confirmation/order summary.

Technical Details

Developed using JavaScript with React.js for a dynamic front end UI. Applied React Router for page routing and Fetch API to communicate product and order data with back-end. Utilized Node.js and a MySQL database to retrieve and deliver data to client. Also applied Bootstrap 4 for a responsive, clean, and fluid user experience.

Dynamic Duo

A memory match game with Calvin & Hobbes theme.

Live GitHub
Overview

A Calvin and Hobbes themed memory match card game with 18 randomized face-down cards. Click to flip cards and execute logic to check for matches and mismatches. Match all 9 pairs to win, and track your stats along the way.

Technical Details

Designed with pure HTML5/CSS3 for structure, styling, and a responsive layout. Utilized JavaScript and jQuery for DOM creation, to calculate statistics, and to enable card-revealing functionality.
